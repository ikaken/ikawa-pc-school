# パソコン教室のイカワ

水戸市のパソコン教室「パソコン教室のイカワ」の公式Webサイト

## 概要

地域活動団体・個人、地域の中小企業を対象に、実務で役立つノウハウを提供するパソコン教室のWebサイトです。

## ファイル構成

```
/
├── index.html          # トップページ
├── privacy.html        # プライバシーポリシー
├── legal.html          # 特定商取引法に基づく表記
├── CNAME               # カスタムドメイン設定
├── css/
│   └── style.css      # スタイルシート
├── js/
│   └── script.js      # JavaScript
└── img/
    ├── ogp.png        # OGP画像（SNSシェア用）
    └── favicon.png    # ファビコン
```

## 主な機能

- **レスポンシブデザイン**: モバイル・タブレット・デスクトップに対応
- **ハンバーガーメニュー**: モバイル表示時のナビゲーション
- **構造化データ（Schema.org）**: LocalBusiness型のJSON-LD実装
- **OGP対応**: SNSシェア時の画像・テキスト表示
- **アクセシビリティ**: WAI-ARIA属性、スキップリンク、セマンティックHTML
- **SEO最適化**: メタタグ、構造化データ、適切な見出し構造

## 技術スタック

- HTML5
- CSS3（カスタムプロパティ、Flexbox）
- Vanilla JavaScript（ES6+）
