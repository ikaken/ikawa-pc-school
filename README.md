# パソコン教室のイカワ

水戸市のパソコン教室「パソコン教室のイカワ」の公式Webサイト

## 概要

地域活動団体・個人、地域の中小企業を対象に、実務で役立つノウハウを提供するパソコン教室のWebサイトです。

## ファイル構成

```
/
├── index.html          # トップページ
├── privacy.html        # プライバシーポリシー
├── legal.html          # 特定商取引法に基づく表記
├── css/
│   └── style.css      # スタイルシート
├── js/
│   └── script.js      # JavaScript
└── img/               # 画像ファイル用ディレクトリ
```

## 技術スタック

- HTML5
- CSS3（カスタムプロパティ、Flexbox）
- Vanilla JavaScript（ES6+）

## 対応ブラウザ

- Chrome（最新版）
- Firefox（最新版）
- Safari（最新版）
- Edge（最新版）
